# frctl

frctl is a fractal visualiser written in C using raylib, supporting Mandelbrot, Julia and Newton fractals.

# Compilation & Running

> **Requirements:**
- raylib
- CMake >= 3.10
- C compiler (gcc, clang)
- UNIX System (Linux, MacOS, WSL)

```sh
mkdir build
cd build
cmake ..
make
cd ../bin
./frctl
```
